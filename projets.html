Portrait Chinois


Rareté : Commune

Niveau requis : Débutant

Description : Une relique de ma première quête mêlant code et création visuelle. Forgé durant le module d’intégration web du DUT, cet artefact matérialise ma personnalité à travers le prisme d’un “portrait chinois” — une série de sept thèmes répondant à la formule mystique : « Si j’étais…, je serais… ».

Propriétés :

  • Interface one-page
  • Menu intégré
  • Animations déclanchées au défilement
  • Non responsive
